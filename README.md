# Machine Learning na Saúde - Diabetes

Aplicação simples em **Streamlit** para a disciplina de Machine Learning.  
O objetivo é mostrar o fluxo completo:

1. Exploração de dados (EDA)
2. Modelo supervisionado (Random Forest Regressor)
3. Modelo não supervisionado (K-Means)

O dataset usado é o **diabetes** do `scikit-learn`, que já vem padronizado.
